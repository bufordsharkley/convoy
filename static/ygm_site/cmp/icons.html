Download a set of four custom You've Got Mail icons for endless desktop computer fun. The icons are shown below, and you can get the whole set at once by selecting Windows or Macintosh and following the instructions further down the page.



Macintosh (.hqx)     Windows (.zip)


Instructions

Macintosh:
  1. When you download the file, it will unstuff to a folder called "Icons". Select the icon you wish to use. Open this folder, and then select the icon you want to use by click on it once.
  2. Choose "Get Info" from the File menu.
  3. Press the tab key, and a square box will appear around the icon in the corner of the Get Info window. Select "Copy" from the File menu, and then close the Get Info window ("Close" from the File menu).
  4. Select the icon (file, folder, disk, etc.) you want to change by clicking it once. Choose "Get Info" from the File menu.
  5. Press the tab key, and a square box will appear around the icon in the corner of the Get Info window. Select "Paste" from the File menu, and then close the Get Info window ("Close" from the File menu).

    That's it!

 Windows 95:
  1. When you download the .zip file, unzip it to a folder you can easily find.
  2. Right-click the shortcut file whose icon you want to change and select "Properties" from the menu.
  3. Select the "Shortcut" tab, and then select the "Change Icon" button.
  4. Select "Browse", and go to the folder containing the icons which you just unzipped.
  5. Select the icon you want to use, hit OK, and you're all done!
